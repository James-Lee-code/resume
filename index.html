程式語言

自主學習

  1. 前端基礎與互動實作（履歷 / 旅遊網站 / 貪食蛇）

    2. 使用 HTML、CSS、JavaScript 建構個人履歷與旅遊資訊網站，完整實作 RWD 響應式設計， 並透過 SCSS 與 Bootstrap 建立可維護的樣式架構。 同時以原生 JavaScript 開發貪食蛇小遊戲，練習 DOM 操作、事件監聽與基礎遊戲邏輯， 強化前端互動與流程控制能力。

  2. MERN Stack 全端專案

    3. 使用 React 作為前端核心，搭配 Node.js + Express 建立後端 API，並以 MongoDB 進行資料儲存。 實作使用者註冊 / 登入 / 留言與 CRUD 功能，透過 RESTful API 串接前後端資料， 並加入表單驗證與錯誤處理機制。前端部署於 Netlify，後端部署於 Render， 完成完整專案開發到上線流程。

  3. React 專案 - 點餐平台 ( React / MongoDB)

    4. 功能：菜單列表顯示、新增菜單、結帳系統、購物車。

  4. React 專案 - Todo 清單

    5. 功能：待辦事項新增、刪除、狀態切換。

  5. Vue 專案 - 課程平台 ( Vue / Firebase )

    6. 功能：課程列表展示、課程內容頁面切換功能、新增課程、登入驗證、留言系統。

  6. Nuxt 專案 - Todo ( Vue / Supabase )

    7. 功能：登入驗證、頁面權限控管、Todo CRUD 操作。
